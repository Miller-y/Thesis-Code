三个关键指标：
RMSE (均方根误差)：越小代表越接近原始数据。
Smoothness (平滑度，基于一阶差分标准差)：越小代表去噪效果越好（越平滑）。
Correlation (相关系数)：越高代表波形特征（如峰值位置、升降趋势）保留得越好。

### 1. RMSE (均方根误差)
**代码实现：**
```python
rmse = np.sqrt(np.mean((raw_data - filtered_data)**2))
```
**原理解读：**
*   **计算过程**：首先计算原始信号 (`raw_data`) 与滤波后信号 (`filtered_data`) 之间差值的平方，然后取平均值，最后开根号。
*   **数学公式**：
    $$ RMSE = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(x_i - y_i)^2} $$
    其中 $x_i$ 是原始数据点，$y_i$ 是滤波后的数据点，$N$ 是数据点总数。
*   **意义**：它直接衡量了滤波后的数据偏离原始数据的程度。值越小，说明滤波后的曲线在数值上越“贴合”原始曲线（即失真越小）。

### 2. Smoothness (平滑度)
**代码实现：**
```python
smoothness = np.std(np.diff(filtered_data))
```
**原理解读：**
*   **计算过程**：
    1.  `np.diff(filtered_data)`：计算相邻两点之间的差值（一阶差分，即“斜率”或“变化量”）。
    2.  `np.std(...)`：计算这些差值的标准差。
*   **数学公式**：
    令差分序列 $d_i = y_{i+1} - y_i$，则：
    $$ Smoothness = \sigma(d) = \sqrt{\frac{1}{N-1}\sum_{i=1}^{N-1}(d_i - \bar{d})^2} $$
*   **意义**：
    *   噪声通常表现为相邻数据点的剧烈波动（差分值忽大忽小）。
    *   通过计算差分的标准差，衡量的是数据的**抖动程度**。
    *   **值越小**，说明相邻点之间的变化越均匀、平缓，代表去噪效果越好（曲线越光滑）。

### 3. Correlation (相关系数)
**代码实现：**
```python
corr = np.corrcoef(raw_data, filtered_data)[0, 1]
```
**原理解读：**
*   **计算过程**：使用 NumPy 的 `corrcoef` 函数计算皮尔逊积矩相关系数矩阵，然后取出矩阵中 `[0, 1]` 位置的元素（即数组 A 和数组 B 之间的相关性）。
*   **数学公式**：
    $$ r = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum(x_i - \bar{x})^2} \sqrt{\sum(y_i - \bar{y})^2}} $$
*   **意义**：
    *   它衡量的是两个波形的**形状相似度**，是线性相关性的度量。
    *   取值范围在 -1 到 1 之间。
    *   **值越接近 1**，说明滤波后的波形虽然可能变平滑了，但整体的起伏趋势、峰谷位置与原始信号高度一致（即没有发生严重的相位滞后或形状改变）。
